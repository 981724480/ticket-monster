TicketMonster. A JBoss Example

TicketMonster is an online ticketing application that gets you started with JBoss technologies.

Buy tickets now!

Built with JDF

TicketMonster's architecture uses the best of the JBoss Developer Framework. Learn how easy it is to harness the power of vanilla Java EE, with a choice of three different UI frameworks: HTML5 and JavaScript, Richfaces/JSF and GWT/Errai.

Mobile-ready

When you interact with TicketMonster from a mobile device, you will get a mobile-specific interface. Learn how easy it is to write applications that can accommodate desktop and mobile clients.

Cloud-ready

A deployment of TicketMonster runs continuously on the OpenShift platform. Learn how easy it is to write applications that can deploy locally and in the cloud.
